Fazer um aplicativo utilizando React-Native para Bruce Wayne o (BATMA)

Este aplicativo tera que gerar senhas de diversos caracteres especias, numeros, letra minuscula e maiscula .
E por fim fazer cola copia.

Aplicativo foi pensando na segurança proposto para Bruce Wayne, por ter que sempre estar gerando senhas novas.


Características

   Gerar uma senha forte aleatória;
   Copiar Passe para Área de Transferência;


Tecnologias
   Este projeto foi desenvolvido com as seguintes tecnologias:

   React-Nativo
   Expo
